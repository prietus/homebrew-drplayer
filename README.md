# Homebrew Tap for DrPlayer

Audiophile music player for MPD with dynamic range analysis, MusicBrainz metadata, smart playlists, and more.

## Install

```bash
brew tap prietus/drplayer
brew install --cask drplayer
```

## Requirements

- macOS 14+ (Sonoma)
- [MPD](https://www.musicpd.org/) and [ffmpeg](https://ffmpeg.org/) (installed automatically as dependencies)
